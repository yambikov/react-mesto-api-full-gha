# «Mesto»
Проектная работа в рамках курса «Веб-разработчик»


**Описание**

Интерактивная страница с возможностью добавлять фотографии, удалять их и ставить лайки.


**Технологии**
* HTML
* CSS
* Flexbox
* Grid-layout
* Позиционирование блоков и элементов
* Адаптивная верстка
* БЭМ Nested
* Медиазапросы
* JavaScript
* Webpack
* Работа с API
* React
